Interactive Physical Reasoning
Project Paper
Introduction
There are 40 different block elimination tasks of 4 folds (Basic, Compositional, Noisy and Multi_ball). The goal is to make the red balls drop into the abyss by eliminating some blocks. The gray and black blocks are static, and the blue blocks are dynamic. Only the gray blocks can be eliminated.
Play

Step 1

Set the environment

Run the commands below in your terminal:

conda create -n iphyre python=3.10

conda activate iphyre

pip install numpy pygame pymunk

pip install iphyre

Step 2

Run the script

Create a '.py' file and specify your name:

from iphyre.utils import collect_play_all

player_name = 'your_name'

collect_play_all(player_name, max_episode=5, save_path=f'./{player_name}.json')

Step 3

Submit the result

Send the '.json' file to lishiqian2001@gmail.com or any other way you like.
